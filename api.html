distance3d.colliders¶

Colliders used for collision detection with GJK algorithm.

ColliderTree

Wraps multiple colliders that are connected through transformations.

ConvexCollider

Convex collider base class.

Convex

Wraps convex hull of a set of vertices for GJK algorithm.

Box

Wraps box for GJK algorithm.

Mesh

Wraps mesh for GJK algorithm (we assume a convex mesh).

Cylinder

Wraps cylinder for GJK algorithm.

Capsule

Wraps capsule for GJK algorithm.

Sphere

Wraps sphere for GJK algorithm.

distance3d.containment¶

Containment methods compute bounding volumes.

axis_aligned_bounding_box

Compute axis-aligned bounding box (AABB) that contains points.

sphere_aabb

Compute axis-aligned bounding box of sphere.

box_aabb

Compute axis-aligned bounding box of an oriented box.

cylinder_aabb

Compute axis-aligned bounding box of cylinder.

capsule_aabb

Compute axis-aligned bounding box of a capsule.

distance3d.epa¶

Expanding polytope algorithm (EPA) for collision resolution after GJK.

epa_vertices

Expanding Polytope Algorithm (EPA).

epa

Expanding Polytope Algorithm (EPA).

distance3d.gjk¶

Gilbert-Johnson-Keerthi (GJK) for distance calculation of convex shapes.

gjk

Gilbert-Johnson-Keerthi algorithm for distance calculation.

gjk_with_simplex

Gilbert-Johnson-Keerthi (GJK) algorithm for distance calculation.
